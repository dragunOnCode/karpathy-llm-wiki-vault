## 一句话摘要

在 Mac 上，Windows 的 Control 键被替换为 Command 键用于大多数键盘快捷键。

## 内容

前 Windows 用户转用 Mac 时，最常用的对照映射是将 Windows 上的 Control 键替换为 Mac 上的 Command 键（⌘）。例如 Command-C 拷贝、Command-V 粘贴。该断言界定了两平台键盘快捷键体系的主体映射关系，是 Windows 用户快速上手 Mac 快捷键的关键起点。
