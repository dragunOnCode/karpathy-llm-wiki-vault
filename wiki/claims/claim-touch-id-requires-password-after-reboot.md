## 一句话摘要

启动或重启 MacBook Air 时必须先键入密码完成初始登录，此后才能使用触控 ID 登录。

## 内容

触控 ID 虽可实现指纹解锁的便捷体验，但在启动或重启后存在安全限制：首次登录必须通过键入密码完成，完成该初始登录后才可使用触控 ID 解锁 Mac。该机制确保在系统启动阶段不依赖指纹而要求显式密码认证，保留一道启动后的安全防线。该断言明确了触控 ID 适用的边界条件。
