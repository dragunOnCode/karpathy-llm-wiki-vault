## 核心摘要

介绍配备触控 ID 的妙控键盘：触控 ID 既是指纹识别传感器也是电源按钮，可用于开机、解锁、Apple Pay 支付与操作认证；启动或重启后需先键入密码完成初始登录，此后可用触控 ID 登录。顶行功能键提供亮度、调度中心、聚焦搜索、听写/Siri、勿扰模式、媒体控制与音量等快捷键。功能 (Fn) 键可打开字符检视器添加表情符号，也可在"系统设置"中自定义为听写或输入法切换等操作。

## 边界与异常

- 触控 ID 解锁需先完成一次密码初始登录（启动/重启后）。
- 功能键行为可能因语言和键盘布局而异。
- Fn 键功能需在"系统设置">"键盘">"按下 Fn 键时"中自定义。
