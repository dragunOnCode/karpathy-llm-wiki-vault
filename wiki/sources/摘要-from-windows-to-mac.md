## 核心摘要

面向前 Windows 用户的 Mac 上手指南，对照解释 Mac 常用术语与操作。访达相当于文件资源管理器；系统设置相当于控制面板；截屏实用工具（Command+Shift+5）相当于截图工具；程序坞相当于开始菜单常用 App。窗口左上角红/黄/绿按钮分别关闭、最小化、全屏。右键点按在 Mac 上称为辅助点按（双指点按或 Control-点按）。键位映射：Windows 的 Ctrl 对应 Mac 的 Command，Alt 对应 Option；Delete 键退格，Fn-Delete 向前删除，Return 等同 Enter。可使用"迁移助理"从 Windows PC 迁移用户设置、文稿、照片、音乐、账户与密码。

## 边界与异常

- Mac 仍可使用许多 Windows 上已有的 App（如 Microsoft Office），可在 App Store 下载。
- 辅助点按方式可在"触控板设置"中自定义。
