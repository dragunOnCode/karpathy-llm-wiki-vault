## 核心摘要

介绍 MacBook Air 的触控 ID（电源）按钮：按下可将 MacBook Air 开机或锁定。设置触控 ID 后，还可将其用于解锁 MacBook Air 和授权购买（如 Apple Pay）。触控 ID 既是指纹识别传感器也是电源按钮。

## 边界与异常

- 启动或重启后需先键入密码完成初始登录，此后才能用触控 ID 登录。
- 触控 ID 设置与添加指纹在"系统设置">"触控 ID"中进行。
