# MacBook Air 最佳提示、技巧和键盘快捷键

在开始使用 MacBook Air 时，你可以尝试这些最佳提示、技巧和键盘快捷键。

| 截屏 / 按下 Command + Shift + 5 ，然后使用屏幕底部的控制截屏或录制视频。截屏或录制视频后，缩略图会出现在屏幕底部。将缩略图拖入文稿或文件夹，向右轻扫以快速保存，或点按以编辑或共享。 | 通过“聚焦”执行操作 / 你可以直接在“聚焦”窗口中完成操作和快捷指令，例如发送信息。按下 Command-空格键 打开“聚焦”，然后按下 Command-3 将搜索范围缩小到仅限操作。输入要查找的内容，或者浏览建议，然后连按要使用的操作。

| 使用触发角 / 通过触发角，你可以将指针移到屏幕一角来快速开始操作。前往“系统设置”，点按“桌面与程序坞” ，然后点按底部的“触发角”。对于要使用的每个角，点按弹出式菜单，然后选取一个选项。 | 通过隔空投送共享项目 / 将照片、视频和文稿等内容无线发送到附近的 iPhone、iPad 或 Mac。点按菜单栏中的 / ![控制中心](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/8a479d4a61f8b310fef4f6fff78fe7a0.png) / ，然后点按“隔空投送” 确保其已打开（应为彩色）。打开要共享的项目，点按工具栏中的 / ![“共享”按钮](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/a8e0b4af388ca65497478b6a1ccd22ec.png) / ，然后点按“隔空投送”。 / > 【注】 两台设备必须打开 Wi-Fi 和蓝牙，且放在距离彼此 10 米（30 英尺）范围内。

| 在电视上镜像 Mac / 你可以使用隔空播放将 Mac 上的内容镜像到电视，或者将电视用作附加显示器。在 Mac 上，点按菜单栏中的 / ![控制中心](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/8a479d4a61f8b310fef4f6fff78fe7a0.png) / ，点按 / ![“镜像”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/9ff47e20fd9301588098fdb6883c7a3d.png) / ，然后选择你的电视。如果询问，请在 Mac 上输入四位数代码。 | 使用 Center Stage 摄像头 / MacBook Air 配备了 [12MP Center Stage 摄像头](https://support.apple.com/zh-cn/guide/macbook-air/aside/apda5f469692/2026/mac/26)，可帮助在使用 FaceTime 通话等视频 App 期间将你及身边人保持在画面中。前往拍摄视频的 App，例如 FaceTime 通话。点按菜单栏中的 / ![视频](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/b02b5247b97886a40a24f441efde3a01.png) / ，然后点按“人物居中”。

| 查找文件 / 从互联网下载的项目通常会保存在“下载”文件夹中，该文件夹位于程序坞右侧。若要查找其他文件的位置，请打开“访达” 窗口。在菜单栏中，选取“显示”>“显示路径栏”。文件的位置会显示在访达窗口的底部。 | 将文件夹添加到“个人收藏” / 你可以在访达的“个人收藏”部分添加“音乐”或“图片”等文件夹。在访达 窗口中，将文件夹拖到边栏中的“个人收藏”下方。

## 键盘快捷键

你可以按下组合键在 Mac 上执行你通常使用触控板、鼠标或其他设备执行的操作。以下是常用键盘快捷键列表。

> 【注】 App 中的键盘快捷键因你在 Mac 上使用的语言和键盘布局而异。如果下方的快捷键未按预期正常工作，请在菜单栏的 App 菜单中查找以查看正确的快捷键。你还可以使用虚拟键盘查看当前键盘布局，称为输入法。请参阅[在 Mac 上使用“虚拟键盘”](https://support.apple.com/zh-cn/guide/mac-help/mchlp1015)。

| 快捷键 | 描述

| Command + X | 剪切所选项目并将其拷贝到剪贴板。

| Command + C | 将所选项目拷贝到剪贴板。

| Command + V | 将剪贴板上的内容粘贴到当前文稿或 App 中。

| Command + Z | 撤销上一个命令。按下 Command-Shift-Z 以重做。

| Command + A | 选择所有项。

| Command + F | 打开“查找”窗口，或者在文稿中查找项目。

| Command + G | 查找当前搜索项目的下一个出现位置。按下 Command-Shift-G 以查找上一个出现位置。

| Command + H | 隐藏最前面 App 的窗口。按下 Command-Option-H 以查看最前面的 App，但是隐藏所有其他 App。

| Command + M | 将当前窗口最小化到程序坞。按下 Command-Option-M 以最小化最前面 App 的所有窗口。

| Command + N | 打开新的文稿或窗口。

| Command + O | 打开所选项目，或者打开对话框以选择要打开的文件。

| Command + P | 打印当前文稿。

| Command + S | 保存当前文稿。

| Command + W | 关闭当前窗口。按下 Command-Option-W 以关闭 App 的所有窗口。

| Command + Q | 退出当前 App。

| Command + Option + Esc | 选取一个 App 以“强制退出”。

| Command + Tab | 在打开的 App 中切换到下一个最常用的 App。
