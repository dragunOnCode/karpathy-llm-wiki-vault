# 从 Windows 转而使用 Mac？

MacBook Air 可完成 Windows 系统所能完成的所有任务，甚至更多。

## 在 Mac 上右键点按

在 Mac 上，右键点按称为辅助点按或按住 Control 键点按。双指点按触控板，或按下键盘上的 Control 键，然后点按一个项目。

如果不喜欢双指点按，你也可以在“触控板设置”中自定义如何执行辅助点按。前往[系统设置](https://support.apple.com/zh-cn/guide/macbook-air/aside/apd796e4d5da/2026/mac/26)，点按边栏中的“触控板”（你可能需要向下滚动），然后从“辅助点按”弹出式菜单中选取一个选项。

## Mac 上的按键和快捷键

许多你熟悉且喜爱的 Windows PC 键盘快捷键在 Mac 上也能使用，只需替换一两个按键即可。

| ![Command 键字形](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/5a2719df0f73b70647f651f9e4d08970.png) / Command / 将 Windows 上的 Control 键替换为 Mac 上用于大多数键盘快捷键的 * Command 键 *。试试看，使用 Command-C 拷贝，使用 Command-V 粘贴。 | ![Option 键](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/ea9aa92ea34bcec877494ba8af70f2e8.png) / Option / 你可以时常将 Windows 上的 Alt 键替换为 Mac 上的 * Option 键 *。使用 Option 键创建特殊字符。例如，若要键入 é，请按下 Option-E，然后再次按下 E 键。

| ![Delete 键](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/3ff048530970178d0aa2e03cc734d5ff.png) / Delete / 在 Mac 上，你可以使用 * Delete 键 *退格，使用 Fn-Delete 向前删除。 | ![Return 键](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/ed18236d1aae80af78bbedf603b24a2d.png) / Return / Mac 上的 * Return 键 *与 Windows PC 上的 Enter 键功能相同。

现在你已熟悉这些按键了，试试几个键盘快捷键。按下 **Command-空格键**打开“聚焦”并搜索 Mac 和网页，或者执行数学问题或货币转换等快速操作。试试 **Command-Shift ****-3** 快速截取整个屏幕，截屏会保存到桌面。或者，如果遇到问题并需要查找 Control-Alt-Delete 的 Mac 等效快捷键，请使用 **Option **** -Command-Esc** 强制退出 App。

有关键盘的更多信息，请参阅[适用于 MacBook Air 的妙控键盘](https://support.apple.com/zh-cn/guide/macbook-air/apdab672d5e9/2026/mac/26)。如需了解更多快捷键，请参阅[键盘快捷键介绍](https://support.apple.com/zh-cn/guide/mac-help/mchld6b9e240)。

## 继续使用你熟悉且喜爱的 App

虽然 Mac 附带了一整套内建 App，可让你保持联络（如“信息”、FaceTime 通话和“电话”）、高效工作（如 Pages 文稿、Keynote 讲演和 Numbers 表格）和娱乐（如 Apple Music 和 Apple TV），但你仍可以使用 Windows PC 上已有的许多 App（如 Microsoft Office）。查看 [App Store](https://support.apple.com/zh-cn/guide/macbook-air/aside/apdc888425fa/2026/mac/26) 下载喜爱的旧 App，甚至可以查找新 App。

## 从 Windows PC 迁移你的内容

设置 MacBook Air 时，你可以从 Windows PC 迁移用户设置、文稿、照片、音乐、账户和密码等内容。若 Mac 已设置好，你仍可迁移内容以及指定要传输的内容。

有关如何迁移内容的更多信息，请参阅[使用“迁移助理”从 PC 传输到 Mac](https://support.apple.com/102565)。
