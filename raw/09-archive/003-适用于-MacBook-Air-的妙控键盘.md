# 适用于 MacBook Air 的妙控键盘

配备触控 ID 的妙控键盘内建许多功能，可让你轻松添加表情符号、更改键盘语言、通过触控 ID 锁定 MacBook Air 以及通过按下某个按键执行许多任务。

![MacBook Air 键盘显示右上方的触控 ID 按钮、左下角的功能键，以及 Tab 键、大写锁定键、Shift 键、Delete 键和 Return 键。](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/c173cb85a3d699bd128902a0bd58838e.png)

## 触控 ID（电源按钮）

触控 ID 是一种指纹识别传感器，同时也是电源按钮。它可用于将 Mac 开机和解锁、使用 Apple Pay 支付购物费用，以及认证某些操作。

| 将 Mac 开机 / 若要将 MacBook Air 开机，请按下触控 ID。你也可以按下 Mac 上的任意按键或点按触控板。 | 设置触控 ID / 若要设置触控 ID 或添加更多指纹，请打开[系统设置](https://support.apple.com/zh-cn/guide/macbook-air/aside/apd796e4d5da/2026/mac/26)并点按边栏中的“触控 ID”。

| 解锁 Mac / 当要求你输入密码时，将手指轻轻放在触控 ID 传感器上。（启动或重启 MacBook Air 时，你需要先键入密码才能登录；在完成该初始登录后，你可以使用触控 ID 登录。） | 使用 Apple Pay 购物 / 若要使用 Apple Pay 购物，请将手指轻轻放在触控 ID 传感器上以完成交易。

你可以使用功能 (Fn) 键 添加表情和符号，或将其设置为执行不同操作，例如开始“听写”或在使用多种语言键入时更改输入法。

## 想要进一步了解？

- 若要快速执行某些任务，你可以使用键盘快捷键，请参阅 Apple 支持文章：[Mac 键盘快捷键](https://support.apple.com/102650)。或者通过[在 Mac 上为 App 创建键盘快捷键](https://support.apple.com/zh-cn/guide/mac-help/mchlp2271)，设置个人自定义键盘快捷键。

- 自定义[“触控 ID 与密码”设置](https://support.apple.com/zh-cn/guide/mac-help/mchl319754b3)以选取可使用触控 ID 执行的操作，或添加额外的指纹。

- 通过[在 Mac 上使用“钱包与 Apple Pay”](https://support.apple.com/zh-cn/guide/mac-help/mchl4773988b)，在 Mac 上设置 Apple Pay。

- 调整[“键盘”设置](https://support.apple.com/zh-cn/guide/mac-help/kbdm162)，如键重复速率、自动文本替换和键盘导航。
