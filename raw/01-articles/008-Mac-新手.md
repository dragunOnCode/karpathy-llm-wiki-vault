# Mac 新手？

如果这是你的第一台 Mac 电脑，这有一些实用信息可帮助你熟悉并开始使用 macOS。

*菜单栏*贯穿 Mac 屏幕顶部，包含菜单、状态图标和控制，便于你快速完成任务。*程序坞*位于屏幕底部，包含你最常使用的 App。*桌面*是菜单栏和程序坞之间的区域，也是你执行大多数操作的地方。桌面上的背景图片称为*墙纸*。

## 使用程序坞和菜单栏

使用菜单栏中的菜单和图标打开 Apple 菜单或控制中心、通过“聚焦”搜索、激活 Siri、查看摄像头和麦克风的状态等。你可以直接在程序坞中点按来打开 App。

![显示菜单栏、程序坞和桌面的 Mac 屏幕。菜单栏中标注了苹果菜单、菜单栏、桌面、“聚焦”图标、Siri 图标、控制中心和程序坞。](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/762f88fecff74895b85fdf3f88af1ab8.png)

| 打开苹果菜单 / 点按菜单栏左侧的苹果菜单 访问“系统设置”、“最近使用的项目”以及“重新启动”、“关机”或“退出登录”等系统命令。 | 在控制中心快速访问设置 / 点按菜单栏右侧的 / ![“控制中心”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/8a479d4a61f8b310fef4f6fff78fe7a0.png) / 打开控制中心，供你立即访问 Mac 的关键设置，如蓝牙、隔空投送、亮度和音量控制等。

| 用“聚焦”搜索 / “聚焦”可轻松查找 Mac 上的所有内容，包括 App、文件、操作，甚至剪贴板上的内容。点按菜单栏右侧的 / ![“聚焦”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/81e7a450f3436a29c39153dd4948cb52.png) / 或按下键盘上的 Command-空格键，然后键入要搜索的内容。 | 询问 Siri / Siri 是 Mac 内建的智能助理，可帮你完成各项任务。点按菜单栏右侧的 / ![Siri 图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/38f6949184e222338a6f2494d528f99c.png) / （或 / ![Siri 图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/6f07b4288fafd7c6189c19f440ead219.png) / ）。或者如果 Apple 智能已打开，按下 Command-Command（按两下）通过键入使用 Siri。

| 查看摄像头和麦克风状态 / 菜单栏中的控制中心旁边出现绿色或橙色指示灯时，表示 摄像头 / ![绿色摄像头指示器](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/4eaa1a8a4179bc8a82294e1809818e2e.png) / 或 麦克风 / ![橙色麦克风指示器](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/536cb52013cb06983be61777d9d65499.png) / 正在使用中。 | 使用程序坞打开 App / 点按程序坞中的 App 将其打开，或点按程序坞中的 / ![App](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/2ff5b782a7fc889d7ca59bb2a450397f.png) / ，然后从出现的 App 列表中点按要打开的 App。程序坞中打开的 App 图标下面有一个黑色圆点。

## 查看和管理窗口

你可以为一个 App 同时打开多个窗口。使用窗口左上角的按钮来调整窗口大小和关闭窗口。

![App 窗口左上角的红色（关闭）按钮、黄色（最小化）按钮和绿色（全屏）按钮。](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/6fc82f986c9d8856b6f025b67db76aaa.png)

| 关闭窗口 / 点按窗口左上角的 / ![“关闭窗口”按钮](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/61130efd9b93e64d820811a21e6688fc.png) / 将其关闭。 | 最小化窗口 / 点按 / ![“最小化窗口”按钮](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/a67fd3af153341eaa9731a41e72b3a60.png) / 将窗口从桌面移到程序坞。

| 更改窗口大小 / 点按 / ![“全屏幕窗口”按钮](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/cd6ddaf8d8999c7a032bf6d897f5db90.png) / 在全屏和非全屏之间切换。你还可以拖移窗口的边缘调整其大小。 | 排列窗口 / 将指针移到 / ![“全屏幕窗口”按钮](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/cd6ddaf8d8999c7a032bf6d897f5db90.png) / 上，然后点按选项将窗口平铺到屏幕的特定部分。

| 查看所有打开的窗口 / 按下妙控键盘顶行中的 / ![“调度中心”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/9e92228e5a839586f3b04da542e9c82f.png) / 进入调度中心，调度中心会将桌面上所有打开的窗口排列成单层，以便轻松找到所需的窗口。 | 退出 App / 关闭 App 窗口时，App 本身仍会在后台运行。若要关闭所有 App 窗口并停止 App 在后台运行，请按住 Control 键点按程序坞中的 App 图标，然后选取“退出”。

## 查找和整理文件

使用访达查找和整理 Mac 上的几乎所有内容，包括文稿、图像、影片及你拥有的任何其他文件。

![打开的访达窗口，以画廊模式显示文件和文件夹。标记按钮和文件详细信息位于右侧。](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/a95f78c99243a0988510168abcb77870.png)

| 打开访达 / 点按程序坞中的 / ![“访达”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/973da250f19955e1602c94cc01b9c95f.png) / 打开访达窗口。左侧的边栏显示你经常使用的项目和文件夹。 | 更改项目的显示方式 / 点按访达窗口顶部的其中一个视图按钮以图标 / ![“图标”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/82ef043716ccabd489bd1f523fc297fe.png) / 、列表 / ![“列表”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/905f05f28a03057c26d397168ffdd79d.png) / 、分栏 / ![“分栏”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/a48ec7c9c932e1d595d44a31d2621175.png) / 或画廊视图 / ![“画廊”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/0255805894a2d46819975ecdb2799114.png) / 查看项目。

| 快速查看项目 / 在访达中选择文件，然后按下空格键查看项目而不打开文件。 | 快速旋转、标记和裁剪文件 / 在分栏 或画廊 视图中，点按该文件，然后点按右侧的 / ![“标记”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/a4a0abe7e5321eb074fb81c9838bb2ee.png) / 查看旋转、标记或创建 PDF 等快速操作，具体取决于所选拼贴的类型。

## 想要进一步了解？

- 若要进一步了解键盘和触控板，包括 Command 键和功能键、常用键盘快捷键和常用手势的相关信息，请参阅[适用于 MacBook Air 的妙控键盘](https://support.apple.com/zh-cn/guide/macbook-air/apdab672d5e9/2026/mac/26)和 [MacBook Air 触控板](https://support.apple.com/zh-cn/guide/macbook-air/apdbb563a1bc/2026/mac/26)。

- 若要进一步了解如何在桌面上移动和排列 App 窗口，请参阅[在 Mac 上移动和排列 App 窗口](https://support.apple.com/zh-cn/guide/mac-help/mchlp2469)、[在分屏浏览视图中使用 App](https://support.apple.com/zh-cn/guide/mac-help/mchl4fbe2921) 和[平铺窗口](https://support.apple.com/zh-cn/guide/mac-help/mchlef287e5d)。

- 如需了解管理所有已打开窗口的不同方式，请参阅[在调度中心中查看打开的窗口和空间](https://support.apple.com/zh-cn/guide/mac-help/mh35798)、[操作多个空间](https://support.apple.com/zh-cn/guide/mac-help/mh14112)和[使用台前调度整理 Mac 桌面](https://support.apple.com/zh-cn/guide/mac-help/mchl534ba392)。

- 若要进一步了解控制中心和通知中心，请参阅[在 Mac 上使用控制中心](https://support.apple.com/zh-cn/guide/mac-help/mchl50f94f8f)和[在 Mac 上使用通知中心](https://support.apple.com/zh-cn/guide/mac-help/mchl2fb1258f)。

- 若要从 Windows 转用，请参阅[从 Windows 转而使用 Mac？](https://support.apple.com/zh-cn/guide/macbook-air/apdd88fdad7a/2026/mac/26)

- 如果有 iPhone，请参阅[使用“连续互通”在 Apple 设备间工作](https://support.apple.com/zh-cn/guide/mac-help/mchl1d734309)，了解如何将 iPhone 与新 Mac 配合使用。
