# 个性化 Mac 的外观

你可以通过多种方式个性化 MacBook Air，从更改 macOS 的外观和使用体验，自定义桌面、程序坞和菜单栏，到调整文本和指针大小。了解如何让 Mac 适合你使用。

![已打开自定义图标、小组件、文件夹和“聚焦”搜索的 Mac 桌面。](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/b3943162c54430056f170200bd9cfaa7.png)

## 调整 macOS 的外观和使用体验

你可以更改 macOS 的外观。例如，切换到深色模式、更改图标和小组件的样式、自定义文件夹的默认颜色等。

| 使用浅色或深色外观 / 前往[系统设置](https://support.apple.com/zh-cn/guide/macbook-air/aside/apd796e4d5da/2026/mac/26)，点按“外观” ，然后选取“自动”、“浅色”或“深色”。选取“自动”时，外观会基于设定的“夜览”定时从浅色切换为深色。 | 更改颜色主题 / 选取用于按钮、弹出式菜单和其他 UI 控制的颜色。前往“系统设置”，点按“外观” ，然后选取“颜色”旁边的选项。

| 更改图标与小组件的外观 / 前往“系统设置”，点按“外观” ，然后选取“图标与小组件样式”旁边的选项。 | 自定义文件夹的默认颜色 / 前往“系统设置”，点按“外观” ，然后选取“文件夹颜色”旁边的选项。

## 自定义桌面

更改墙纸、自定义屏幕保护程序或添加文件夹，使桌面专属于你。你还可以添加小组件，便于你速览日历或提醒事项等信息。

| 更换墙纸 / 前往[系统设置](https://support.apple.com/zh-cn/guide/macbook-air/aside/apd796e4d5da/2026/mac/26)，点按“墙纸” ，然后选取想要的墙纸，或点按“添加照片”选取自己的照片。 | 自定义屏幕保护程序 / 前往“系统设置”，点按“墙纸” ，然后点按“屏幕保护程序”。选取“自定义”选择自己的屏幕保护程序。

| 添加或编辑小组件 / 若要添加小组件，请按住 Control 键点按桌面，然后选取“编辑小组件”。使用边栏浏览不同的小组件。将新的小组件拖到桌面上的任意位置或拖入通知中心。若要编辑，请按住 Control 键点按小组件，然后从可用选项中选取，例如“小”、“中”或“大”。 | 添加自定义文件夹 / 按住 Control 键点按桌面，然后选取“新建文件夹”。你可以随后更改其颜色和添加图标或表情符号来自定义文件夹：按住 Control 键点按文件夹，选取“自定义文件夹”，然后尝试直到你满意文件夹的外观。

## 自定义程序坞和菜单栏

调整程序坞的大小、将它移到屏幕左侧或右侧、将它设为不使用时隐藏等。你还可以从菜单栏和控制中心添加或移除项目。

| 更改程序坞的外观 / 前往[系统设置](https://support.apple.com/zh-cn/guide/macbook-air/aside/apd796e4d5da/2026/mac/26)，点按“桌面与程序坞” ，然后尝试不同的选项，例如增加 App 的大小或移动程序坞位置。 | 在程序坞中添加或移除项目 / 若要将项目添加到程序坞，请打开访达窗口，然后将要添加的项目拖入程序坞。若要移除项目，请将其拖到程序坞中的废纸篓图标直至看到“从程序坞中移除”。

| 将常用项目添加到菜单栏 / 将状态菜单添加到菜单栏右侧，以便快速查看和调整设置，如 Wi-Fi 或勿扰模式。前往“系统设置”，点按“菜单栏” ，然后在“菜单栏控制”下方选择项目。 | 自定义“控制中心” / 点按 / ![“控制中心”图标](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/8a479d4a61f8b310fef4f6fff78fe7a0.png) / ，然后点按“编辑控制”。在出现的窗口中，将指针移到要添加的控制上，然后点按左上角的 / ![“添加”按钮](https://help.apple.com/assets/68E55784E266DE1F4A0ACA56/69726E59320F229D6106C907/zh_CN/1fd6a5da6fcf6102183f42ec157d4161.png) / 将其添加到控制中心。

## 更改文本和指针大小

更改多个 App 间和桌面上的文本大小，以及增加指针大小使其更易看清。

| 增加文本大小 / 前往[系统设置](https://support.apple.com/zh-cn/guide/macbook-air/aside/apd796e4d5da/2026/mac/26)，点按“辅助功能” ，然后点按“显示”。点按“文字大小”旁边的“默认”，然后调整滑块至首选文字大小。 | 增加指针大小 / 前往“系统设置”，点按“辅助功能” ，点按“显示”，然后调整“指针大小”旁边的滑块。

| 增加边栏中的文本和图标大小 / 前往“系统设置”，点按“外观” ，然后从“边栏图标大小”旁边的弹出式菜单中选取一个选项。

## 想要进一步了解？

- 若要自定义菜单栏、程序坞和控制中心，请参阅[自定义菜单栏](https://support.apple.com/zh-cn/guide/mac-help/mchl4af84660)、[使用程序坞](https://support.apple.com/zh-cn/guide/mac-help/mh35859)和[使用控制中心](https://support.apple.com/zh-cn/guide/mac-help/mchl50f94f8f)。

- 有关不同辅助功能（包括视觉、听觉、活动、语音和认知）的信息，请参阅[开始使用辅助功能](https://support.apple.com/zh-cn/guide/macbook-air/apd4c143b3c5/2026/mac/26)。

- 若要了解如何使用键盘快捷键和触控板手势，请参阅[键盘快捷键](https://support.apple.com/zh-cn/guide/macbook-air/apde5ace2f6f/2026/mac/26#apd604f920e2)和[常用触控板手势](https://support.apple.com/zh-cn/guide/macbook-air/apdbb563a1bc/2026/mac/26#apd1cb71c216)。

- 许多 App 窗口都包含带按钮的工具栏，可供访问常用功能、工具或操作。若要了解如何自定义工具栏，请参阅[在 Mac 上自定义工具栏](https://support.apple.com/zh-cn/guide/mac-help/mchlb294260e)。

- 若要了解如何更改接收的通知，请参阅[在 Mac 上更改“通知”设置](https://support.apple.com/zh-cn/guide/mac-help/mchl205da693)。你还可以[设置专注模式](https://support.apple.com/zh-cn/guide/mac-help/mchl613dc43f)暂停和静音所有或特定通知，从而减少干扰并完成任务。
